Triển khai ARANID thành website thật

Bước 1 – Tạo file

Dùng Acode hoặc SPCK trên điện thoại → tạo file index.html → dán toàn bộ code.

Bước 2 – Upload GitHub

Tạo repository mới → upload index.html → vào Settings → Pages → bật GitHub Pages.

Bước 3 – Nhận link

Bạn sẽ có link dạng: https://tenban.github.io/aranid để chia sẻ hoặc trình diễn.
